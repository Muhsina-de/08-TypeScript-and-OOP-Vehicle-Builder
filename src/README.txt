# Vehicle Builder App

The Vehicle Builder App is designed to help users create and manage a variety of vehicles through a user-friendly interface. By leveraging TypeScript and Object-Oriented Programming (OOP) principles, the app ensures that the codebase is modular, maintainable, and scalable. Users can define vehicle attributes such as make, model, year, and color, and perform operations to manage these vehicles efficiently.

## Overview
The Vehicle Builder App is a TypeScript-based application that allows users to create and manage different types of vehicles using Object-Oriented Programming (OOP) principles.

## Features
- Create various types of vehicles (e.g., cars, trucks, motorcycles)
- Manage vehicle properties such as make, model, year, and color
- Utilize OOP principles to ensure modularity and reusability of code

## Installation
1. Clone the repository:
    ```
    git clone https://github.com/yourusername/vehicle-builder-app.git
    ```
2. Navigate to the project directory:
    ```
    cd vehicle-builder-app
    ```
3. Install the dependencies:
    ```
    npm install
    ```

## Usage
1. Build the project:
    ```
    npm run build
    ```
2. Run the application:
    ```
    npm start
    ```

## Project Structure
- `src/`: Contains the TypeScript source files
- `dist/`: Contains the compiled JavaScript files
- `test/`: Contains the test files
